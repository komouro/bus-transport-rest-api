Weather Conditions

Find the bus lines that have a source-station with no raining conditions.


You need to specify the type of station you are interested in (station type) and the source of the information (service type). If the option "source" is chosen, the server returns the stations where it is not raining, along with the bus lines that can be used from these stations. If the option "destination" is chosen, the server returns the stations that are reachable from points where it is not raining, along with the bus lines that can be used to access these stations.
